# Understanding Artificial Intelligence

Artificial Intelligence (AI) is the simulation of human intelligence processes by computer systems. These processes include learning, reasoning, problem-solving, perception, and language understanding.

---

## 💡 Core Subfields of AI

*   **Machine Learning (ML):** Systems that learn from data to improve performance without explicit programming.
*   **Deep Learning:** A subset of ML utilizing multi-layered artificial neural networks.
*   **Natural Language Processing (NLP):** Technology enabling machines to read, understand, and derive meaning from human languages.
*   **Computer Vision:** Algorithms that allow computers to extract meaningful information from digital images or videos.

---

## ⚖️ Ethics and Challenges

AI development comes with significant responsibilities. Key challenges include:

1.  **Data Bias:** Algorithms can perpetuate human biases present in training datasets.
2.  **Privacy:** Large language models require massive data collection, threatening personal privacy.
3.  **Job Displacement:** Automation changes the labor market landscape rapidly.

## 💻 Sample Code: Simple AI Logic (Python)

Below is an example of an basic operational rule-based system in Python:

```python
def simple_ai_agent(input_text):
    # Basic intent classification loop
    intent = input_text.lower()
   
    if "hello" in intent or "hi" in intent:
        return "Greetings! How can I assist you today?"
    elif "weather" in intent:
        return "I recommend checking a live weather API for real-time updates."
    else:
        return "I am learning every day. Can you rephrase that?"

# Execute the agent
print(simple_ai_agent("Hello there!"))
```
